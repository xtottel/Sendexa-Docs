# Phonebook Webhooks

Receive real-time notifications when contacts or groups are created, updated, or deleted.

---

## Webhook Events

### Contact Events

| Event                  | Trigger              |
| ---------------------- | -------------------- |
| `contact.created`      | New contact created  |
| `contact.updated`      | Contact updated      |
| `contact.deleted`      | Contact deleted      |
| `contact.subscribed`   | Contact subscribed   |
| `contact.unsubscribed` | Contact unsubscribed |

### Group Events

| Event                   | Trigger                    |
| ----------------------- | -------------------------- |
| `group.created`         | New group created          |
| `group.updated`         | Group updated              |
| `group.deleted`         | Group deleted              |
| `group.contact_added`   | Contact added to group     |
| `group.contact_removed` | Contact removed from group |

---

## Webhook Payload

### Contact Created

```json
{
  "event": "contact.created",
  "contact": {
    "id": "contact_abc123",
    "phone": "233501234567",
    "email": "john@example.com",
    "first_name": "John",
    "last_name": "Doe",
    "created_at": "2025-03-12T14:20:00Z"
  },
  "timestamp": "2025-03-12T14:20:00Z"
}
```

### Contact Updated

```json
{
  "event": "contact.updated",
  "contact": {
    "id": "contact_abc123",
    "updated_fields": ["first_name", "custom_fields"],
    "updated_at": "2025-03-12T14:25:00Z"
  },
  "timestamp": "2025-03-12T14:25:00Z"
}
```

### Group Contact Added

```json
{
  "event": "group.contact_added",
  "group_id": "group_xyz789",
  "contact_id": "contact_abc123",
  "timestamp": "2025-03-12T14:30:00Z"
}
```

## Implementing Webhook Handler

### Node.js

```javascript
const express = require('express');
const app = express();

app.post('/webhooks/phonebook', (req, res) => {
  const event = req.body;

  switch (event.event) {
    case 'contact.created':
      handleContactCreated(event.contact);
      break;

    case 'contact.unsubscribed':
      handleUnsubscribe(event.contact);
      break;
  }

  res.status(200).send('OK');
});

async function handleContactCreated(contact) {
  console.log('New contact:', contact.id);
  // Send welcome message
  await sendSMS(contact.phone, 'Welcome!');
}
```
