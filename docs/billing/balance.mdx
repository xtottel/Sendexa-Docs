# Check Balance

Monitor your Sendexa credit balance programmatically or through the dashboard.

---

## API Endpoint

### GET /v2/billing/balance

Retrieve your current credit balance and account information.

**Base URL:**
```
https://smsc.sendexa.co
```

---

## Request Example

### JavaScript (Node.js)

```js
import axios from 'axios';

async function checkBalance() {
  const res = await axios.get(
    'https://smsc.sendexa.co/v2/billing/balance',
    {
      headers: {
        Authorization: `Bearer YOUR_TOKEN_HERE`,
        'Content-Type': 'application/json',
      },
    }
  );

  console.log(res.data);
}

checkBalance();
```

### cURL

```bash
curl -X GET https://smsc.sendexa.co/v2/billing/balance \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json"
```

### Python

```python
import requests

url = "https://smsc.sendexa.co/v2/billing/balance"
headers = {
    "Authorization": "Bearer YOUR_TOKEN_HERE",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
```

---

## Response Structure

### Success Response

```json
{
  "status": "success",
  "balance": 150.50,
  "currency": "GHS",
  "account_status": "active",
  "last_updated": "2025-03-12T10:30:00Z"
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Request status (`success` or `error`) |
| `balance` | float | Current credit balance |
| `currency` | string | Currency code (GHS, USD, etc.) |
| `account_status` | string | Account status (`active`, `suspended`, etc.) |
| `last_updated` | string | Last balance update timestamp (ISO 8601) |

---

## Error Responses

### 401 Unauthorized

```json
{
  "status": "error",
  "message": "Invalid or missing authorization token"
}
```

### 403 Forbidden

```json
{
  "status": "error",
  "message": "Account suspended. Please contact support."
}
```

## Check Balance in Dashboard

**Manual Check:**
1. Log in to your dashboard
2. View balance in top navigation bar
3. Click **Billing** for detailed breakdown
