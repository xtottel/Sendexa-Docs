## Key Features

### Contact Management

Create, update, and manage individual contacts:

- Store phone numbers, emails, names
- Add custom fields
- Track subscription status
- Manage opt-in/opt-out
- View contact history

**Endpoint:** `POST /v1/phonebook/contacts`

```json
{
  "phone": "233501234567",
  "email": "user@example.com",
  "first_name": "John",
  "last_name": "Doe",
  "custom_fields": {
    "birthday": "1990-01-15",
    "city": "Accra"
  }
}
```

### Group Management

Organize contacts into groups for targeted campaigns:

- Create multiple groups
- Add/remove contacts
- Nested group structure
- Group-level permissions
- Track group size

**Endpoint:** `POST /v1/phonebook/groups`

```json
{
  "name": "Premium Customers",
  "description": "Customers with premium subscription",
  "tags": ["premium", "active"]
}
```

### Bulk Operations

Import or update thousands of contacts at once:

- CSV import
- Bulk create/update
- Batch delete
- Mass group assignment
- Progress tracking

**Endpoint:** `POST /v1/phonebook/contacts/bulk`

### Smart Segments

Create dynamic groups based on criteria:

- Filter by custom fields
- Date-based segments
- Behavior-based groups
- Automatic updates
- Real-time counts

## Getting Started

### 1. Create an Account

Sign up at [app.sendexa.co](https://app.sendexa.co) and verify your email.

### 2. Generate API Token

1. Navigate to **Settings ‚Üí API Tokens**
2. Click **Create New Token**
3. Copy your token securely
4. Use it in the `Authorization` header

```http
Authorization: Bearer sk_live_your_token_here
```

### 3. Create Your First Contact

```bash
curl -X POST 'https://api.sendexa.co/v1/phonebook/contacts' \
  -H 'Authorization: Bearer sk_live_your_token' \
  -H 'Content-Type: application/json' \
  -d '{
    "phone": "233501234567",
    "email": "john@example.com",
    "first_name": "John",
    "last_name": "Doe"
  }'
```

### 4. Create a Group

```bash
curl -X POST 'https://api.sendexa.co/v1/phonebook/groups' \
  -H 'Authorization: Bearer sk_live_your_token' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Newsletter Subscribers",
    "description": "Users subscribed to newsletter"
  }'
```

### 5. Add Contact to Group

```bash
curl -X POST 'https://api.sendexa.co/v1/phonebook/groups/{group_id}/contacts' \
  -H 'Authorization: Bearer sk_live_your_token' \
  -H 'Content-Type: application/json' \
  -d '{
    "contact_id": "contact_abc123"
  }'
```

```json
{

  "email": "john@example.com",
  "first_name": "John",
```

### Field Descriptions

| Field           | Type    | Description                         |
| --------------- | ------- | ----------------------------------- |
| `id`            | string  | Unique contact identifier           |
| `phone`         | string  | Phone number (international format) |
| `email`         | string  | Email address                       |
| `first_name`    | string  | First name                          |
| `last_name`     | string  | Last name                           |
| `status`        | string  | active/inactive/blocked             |
| `subscribed`    | boolean | Subscription status                 |
| `custom_fields` | object  | Custom attributes                   |
| `tags`          | array   | Contact tags                        |
| `groups`        | array   | Group IDs                           |
| `created_at`    | string  | Creation timestamp                  |
| `updated_at`    | string  | Last update timestamp               |

## Common Use Cases

### 1. Newsletter Management

```javascript
// Create newsletter group
const group = await createGroup({
  name: 'Newsletter Subscribers',
  description: 'Users who opted in to newsletter',
});

// Add subscriber
async function subscribeToNewsletter(email, name) {
  // Create contact
  const contact = await createContact({
    email,
    first_name: name,
    subscribed: true,
    tags: ['newsletter'],
  });

  // Add to group
  await addContactToGroup(group.id, contact.id);

  return contact;
}
```

### 2. Customer Segmentation

```javascript
// Create segments
const segments = {
  premium: await createGroup({ name: 'Premium Customers' }),
  active: await createGroup({ name: 'Active Users' }),
  inactive: await createGroup({ name: 'Inactive Users' }),
};

// Segment customers
async function segmentCustomer(customer) {
  const contact = await createContact({
    phone: customer.phone,
    email: customer.email,
    first_name: customer.name,
    custom_fields: {
      plan: customer.plan,
      last_purchase: customer.lastPurchase,
    },
  });

  // Add to appropriate segment
  if (customer.plan === 'premium') {
    await addContactToGroup(segments.premium.id, contact.id);
  }

  const daysSinceLastPurchase = getDaysSince(customer.lastPurchase);
  if (daysSinceLastPurchase < 30) {
    await addContactToGroup(segments.active.id, contact.id);
  } else {
    await addContactToGroup(segments.inactive.id, contact.id);
  }

  return contact;
}
```

```javascript
  const today = new Date();
  const currentMonth = today.getMonth() + 1;

  // Get all contacts
  const contacts = await listContacts();

  // Filter by birthday month
  for (const contact of contacts) {
    if (contact.custom_fields?.birthday) {
      const birthday = new Date(contact.custom_fields.birthday);

      if (birthday.getMonth() + 1 === currentMonth) {
        await addContactToGroup(birthdayGroup.id, contact.id);
      }
    }
  }

  // Send birthday messages
  const birthdayContacts = await getGroupContacts(birthdayGroup.id);
  for (const contact of birthdayContacts) {
    const birthday = new Date(contact.custom_fields.birthday);
    if (birthday.getDate() === today.getDate()) {
      await sendSMS(contact.phone, `Happy Birthday ${contact.first_name}! üéâ`);
    }
  }
}
```

## Opt-In/Opt-Out Management

### Managing Subscriptions

```javascript
// Opt-in
async function optIn(phone) {
  const contact = await findOrCreateContact(phone);

  await updateContact(contact.id, {
    subscribed: true,
    custom_fields: {
      opt_in_date: new Date().toISOString(),
      opt_in_source: 'web_form',
    },
  });

  // Send confirmation
  await sendSMS(
    phone,
    'You have subscribed to our updates. Reply STOP to unsubscribe.'
  );
}

// Opt-out
async function optOut(phone) {
  const contact = await getContactByPhone(phone);

  if (contact) {
    await updateContact(contact.id, {
      subscribed: false,
      custom_fields: {
        opt_out_date: new Date().toISOString(),
      },
    });

    await sendSMS(
      phone,
      'You have been unsubscribed. Reply START to resubscribe.'
    );
  }
}

// Check subscription
async function isSubscribed(phone) {
  const contact = await getContactByPhone(phone);
  return contact?.subscribed || false;
}
```

## Best Practices

### 1. Deduplication

```javascript
// Check for duplicates before creating
async function createContactSafe(data) {
  // Check by phone
  let contact = await getContactByPhone(data.phone);

  if (contact) {
    // Update existing
    return await updateContact(contact.id, data);
  }

  // Check by email if provided
  if (data.email) {
    contact = await getContactByEmail(data.email);
    if (contact) {
      return await updateContact(contact.id, data);
    }
  }

  // Create new
  return await createContact(data);
}
```

### 2. Batch Operations

```javascript
// Bad - Individual requests
for (const contact of contacts) {
  await createContact(contact);
}

// Good - Bulk import
await bulkCreateContacts(contacts);
```

### 3. Error Handling

```javascript
async function safeCreateContact(data) {
  try {
    return await createContact(data);
  } catch (error) {
    if (error.code === 'DUPLICATE_PHONE') {
      // Update instead
      const existing = await getContactByPhone(data.phone);
      return await updateContact(existing.id, data);
    }
    throw error;
  }
}
```

### 4. Group Size Monitoring

```javascript
// Monitor group sizes
async function checkGroupLimits() {
  const groups = await listGroups();

  for (const group of groups) {
    if (group.contact_count > 10000) {
      console.warn(
        `Large group: ${group.name} (${group.contact_count} contacts)`
      );
    }
  }
}
```

### 5. Data Validation

```javascript
// Validate before creating
function validateContact(data) {
  const errors = [];

  if (!data.phone && !data.email) {
    errors.push('Phone or email required');
  }

  if (data.phone && !isValidPhone(data.phone)) {
    errors.push('Invalid phone number');
  }

  if (data.email && !isValidEmail(data.email)) {
    errors.push('Invalid email address');
  }

  return errors;
}

async function createValidatedContact(data) {
  const errors = validateContact(data);

  if (errors.length > 0) {
    throw new Error(`Validation failed: ${errors.join(', ')}`);
  }

  return await createContact(data);
}
```
