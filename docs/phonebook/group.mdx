# Group Management

Organize contacts into groups for targeted campaigns. Create groups, add/remove contacts, and manage group memberships.

---

## Quick Start

### Create Group

```bash
curl -X POST 'https://api.sendexa.co/v1/phonebook/groups' \
  -H 'Authorization: Bearer sk_live_your_token' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Premium Customers",
    "description": "Customers with premium subscription"
  }'
```

### Add Contact to Group

```bash
curl -X POST 'https://api.sendexa.co/v1/phonebook/groups/group_xyz789/contacts' \
  -H 'Authorization: Bearer sk_live_your_token' \
  -H 'Content-Type: application/json' \
  -d '{
    "contact_id": "contact_abc123"
  }'
```

## Create Group

### POST /v1/phonebook/groups

```javascript
const group = await axios.post(
  'https://api.sendexa.co/v1/phonebook/groups',
  {
    name: 'Premium Customers',
    description: 'High-value customers',
    tags: ['premium', 'vip'],
  },
  {
    headers: {
      Authorization: 'Bearer sk_live_your_token',
    },
  }
);
```

#### Response

```json
{
  "success": true,
  "group": {

    "tags": ["premium", "vip"],
    "created_at": "2025-03-12T14:20:00Z"
  }
}
```

## Common Use Cases

### 1. Campaign Segmentation

```javascript
// Create campaign groups
const groups = {
  engaged: await createGroup({ name: 'Engaged Users' }),
  inactive: await createGroup({ name: 'Inactive Users' }),
  vip: await createGroup({ name: 'VIP Customers' }),
};

// Segment contacts
async function segmentContacts() {
  const contacts = await listContacts();

  for (const contact of contacts) {
    const daysSinceLastPurchase = getDaysSince(
      contact.custom_fields.last_purchase
    );

    if (daysSinceLastPurchase < 30) {
      await addToGroup(groups.engaged.id, contact.id);
    } else if (daysSinceLastPurchase > 90) {
      await addToGroup(groups.inactive.id, contact.id);
    }

    if (contact.custom_fields.lifetime_value > 1000) {
      await addToGroup(groups.vip.id, contact.id);
    }
  }
}
```

### 2. Event Management

```javascript
// Create event group
const eventGroup = await createGroup({
  name: 'Conference 2025',
  description: 'Registered attendees',
  tags: ['event'],
});

// Register attendee
async function registerAttendee(email, phone, name) {
  const contact = await createContact({
    email,
    phone,
    first_name: name,
    custom_fields: {
      event: 'Conference 2025',
      registration_date: new Date().toISOString(),
    },
  });

  await addToGroup(eventGroup.id, contact.id);

  // Send confirmation
  await sendSMS(phone, 'Registration confirmed!');
}
```
