# WhatsApp Broadcasts

Send WhatsApp messages to multiple contacts at once. Perfect for marketing campaigns, announcements, and notifications to segmented audiences.

---

## Overview

### What are Broadcasts?

Broadcasts allow you to send the same message to multiple recipients in a single API call. Each recipient receives an individual message, maintaining personal engagement.

### Key Features

- **Bulk Sending** - Message thousands at once
- **Contact Targeting** - Send to specific segments
- **Delivery Tracking** - Monitor individual delivery status
- **Cost Effective** - Optimized conversation management
- **Template Support** - Use approved templates

## Quick Start

```bash

      "type": "template",

        "name": "promotion_alert",

      }
    }
  }'
```

## Endpoint

### POST /v1/whatsapp/broadcasts

Create and send a broadcast campaign.

**Base URL:** `https://api.sendexa.co`

**Rate Limit:** 20 requests per minute

---

## Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | Broadcast campaign name |
| `contacts` | array | Yes* | Array of phone numbers |
| `contact_ids` | array | Yes* | Array of contact IDs from contact management |
| `tags` | array | No | Send to contacts with these tags |
| `message` | object | Yes | Message object (same as send endpoint) |
| `schedule_at` | string | No | ISO 8601 timestamp for scheduling |

*Either `contacts`, `contact_ids`, or `tags` required

### Request Example


## Code Examples

### JavaScript / Node.js

```javascript
const axios = require('axios');

class BroadcastManager {
  constructor(apiToken) {
    this.apiToken = apiToken;
    this.baseUrl = 'https://api.sendexa.co/v1/whatsapp';
  }

  async createBroadcast(name, recipients, message) {
    try {
      const response = await axios.post(
        `${this.baseUrl}/broadcasts`,
        {
          name,
          contacts: recipients,
          message
        },
        {
          headers: {
            'Authorization': `Bearer ${this.apiToken}`,
            'Content-Type': 'application/json'
          }
        }
      );

      console.log(`Broadcast created: ${response.data.broadcast.id}`);
      return response.data.broadcast;
    } catch (error) {
      console.error('Broadcast error:', error.response?.data || error.message);
      throw error;
    }
  }

  async getBroadcastStatus(broadcastId) {
    const response = await axios.get(
      `${this.baseUrl}/broadcasts/${broadcastId}`,
      {
        headers: {
          'Authorization': `Bearer ${this.apiToken}`
        }
      }
    );

    return response.data.broadcast;
  }

  async waitForCompletion(broadcastId, maxWait = 300000) {
    const start = Date.now();
    
    while (Date.now() - start < maxWait) {
      const broadcast = await this.getBroadcastStatus(broadcastId);
      
      if (broadcast.status === 'completed') {
        return broadcast;
      } else if (broadcast.status === 'failed') {
        throw new Error('Broadcast failed');
      }
      
      // Wait 5 seconds before checking again
      await new Promise(resolve => setTimeout(resolve, 5000));
    }
    
    throw new Error('Broadcast timeout');
  }
}

// Usage
const manager = new BroadcastManager('sk_live_your_token');

// Create broadcast
const broadcast = await manager.createBroadcast(
  'Flash Sale Alert',
  ['233501234567', '233509876543'],
  {
    type: 'template',
    template: {
      name: 'flash_sale',
      language: 'en'
    }
  }
);

// Wait for completion
const completed = await manager.waitForCompletion(broadcast.id);
console.log('Broadcast completed:', completed.stats);
```

### Python

```python
import requests
import time

class BroadcastManager:
    def __init__(self, api_token):
        self.api_token = api_token
        self.base_url = 'https://api.sendexa.co/v1/whatsapp'
        self.headers = {
            'Authorization': f'Bearer {api_token}',
            'Content-Type': 'application/json'
        }
    
    def create_broadcast(self, name, recipients, message):
        """Create a new broadcast"""
        url = f'{self.base_url}/broadcasts'
        payload = {
            'name': name,
            'contacts': recipients,
            'message': message
        }
        
        response = requests.post(url, json=payload, headers=self.headers)
        response.raise_for_status()
        
        broadcast = response.json()['broadcast']
        print(f"Broadcast created: {broadcast['id']}")
        return broadcast
    
    def get_broadcast_status(self, broadcast_id):
        """Get broadcast status"""
        url = f'{self.base_url}/broadcasts/{broadcast_id}'
        response = requests.get(url, headers=self.headers)
        response.raise_for_status()
        return response.json()['broadcast']
    
    def wait_for_completion(self, broadcast_id, max_wait=300):
        """Wait for broadcast to complete"""
        start = time.time()
        
        while time.time() - start < max_wait:
            broadcast = self.get_broadcast_status(broadcast_id)
            
            if broadcast['status'] == 'completed':
                return broadcast
            elif broadcast['status'] == 'failed':
                raise Exception('Broadcast failed')
            
            time.sleep(5)
        
        raise Exception('Broadcast timeout')

# Usage
manager = BroadcastManager('sk_live_your_token')

# Create broadcast
broadcast = manager.create_broadcast(
    'Flash Sale Alert',
    ['233501234567', '233509876543'],
    {
        'type': 'template',
        'template': {
            'name': 'flash_sale',
            'language': 'en'
        }
    }
)

# Wait for completion
completed = manager.wait_for_completion(broadcast['id'])
print(f"Delivered: {completed['stats']['delivered']}/{completed['stats']['total']}")
```

```php
<?php

class BroadcastManager {
    private $apiToken;
    private $baseUrl = 'https://api.sendexa.co/v1/whatsapp';
    
    public function __construct($apiToken) {
        $this->apiToken = $apiToken;
    }
    
    public function createBroadcast($name, $recipients, $message) {
        $url = $this->baseUrl . '/broadcasts';
        
        $payload = [
            'name' => $name,
            'contacts' => $recipients,
            'message' => $message
        ];
        
        $ch = curl_init($url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
        curl_setopt($ch, CURLOPT_POST, true);
        curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($payload));
        curl_setopt($ch, CURLOPT_HTTPHEADER, [
            'Authorization: Bearer ' . $this->apiToken,
            'Content-Type: application/json'
        ]);
        
        $response = curl_exec($ch);
        $httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
        curl_close($ch);
        
        if ($httpCode === 200) {
            $data = json_decode($response, true);
            echo "Broadcast created: " . $data['broadcast']['id'] . "\n";
            return $data['broadcast'];
        } else {
            throw new Exception("Broadcast creation failed: " . $response);
        }
    }
    
    public function getBroadcastStatus($broadcastId) {
        $url = $this->baseUrl . '/broadcasts/' . $broadcastId;
        
        $ch = curl_init($url);
        curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
        curl_setopt($ch, CURLOPT_HTTPHEADER, [
            'Authorization: Bearer ' . $this->apiToken
        ]);
        
        $response = curl_exec($ch);
        curl_close($ch);
        
        $data = json_decode($response, true);
        return $data['broadcast'];
    }
}

            'language' => 'en'
        ]
    ]
);

// Get status
$status = $manager->getBroadcastStatus($broadcast['id']);
echo "Status: " . $status['status'] . "\n";
```

```javascript
// Send different messages to different segments
const vipCustomers = await getContactsByTag('vip');

await createBroadcast('VIP Exclusive', vipCustomers, vipMessage);
```

### 2. Batch Large Broadcasts

```javascript
// Split large lists into batches
function chunkArray(array, size) {
  const chunks = [];
  for (let i = 0; i < array.length; i += size) {
    chunks.push(array.slice(i, i + size));
  }
  return chunks;
}

const allContacts = await getAllContacts(); // 50,000 contacts
const batches = chunkArray(allContacts, 5000);

for (const [index, batch] of batches.entries()) {
  await createBroadcast(
    `Campaign Batch ${index + 1}`,
    batch,
    message
  );
  
  // Small delay between batches
  await new Promise(r => setTimeout(r, 2000));
}
```

### 3. Monitor Delivery Rates

```javascript
async function monitorBroadcast(broadcastId) {
  const broadcast = await getBroadcastStatus(broadcastId);
  const stats = broadcast.stats;
  
  const deliveryRate = (stats.delivered / stats.total) * 100;
  const failureRate = (stats.failed / stats.total) * 100;
  
  console.log(`Delivery Rate: ${deliveryRate.toFixed(2)}%`);
  console.log(`Failure Rate: ${failureRate.toFixed(2)}%`);
  
  if (failureRate > 5) {
    // Alert if failure rate too high
    await alertAdmin(`High failure rate in broadcast ${broadcastId}`);
  }
  
  return { deliveryRate, failureRate };
}
```

### 5. Test Before Broadcasting


## Common Errors

### Error: Insufficient Balance

```json
{
  "error": "INSUFFICIENT_BALANCE",
  "message": "Not enough credits for broadcast",
  "details": {
    "required": 450.00,
    "available": 200.00,
    "currency": "GHS"
  }
}
```

**Solution:** Top up account before broadcasting.

### Error: Invalid Recipients

```json
{
  "error": "INVALID_RECIPIENTS",
  "message": "Some recipients have invalid phone numbers",
  "details": {
    "invalid_count": 15,
    "invalid_numbers": ["0501234567", "invalid"]
  }
}
```

**Solution:** Validate phone numbers before broadcasting.

### Error: Template Not Approved

```json
{
  "error": "TEMPLATE_NOT_APPROVED",
  "message": "Template 'flash_sale' is pending approval"
}
```

**Solution:** Wait for template approval before broadcasting.
